# myproject_v.0.3

### Prerequisites
- Python 3.6+ (x64). 
- Django 2.0.2+.
- Pillow 5.0+.

Please check your python version with the following command. The result should be 3.6 or higher.
```
python3 --version
```
Django and Pillow can be installed via pip.
```
pip3 isntall -r requirements.txt
```

### Run

#### From folder mysite
```
python3 manage.py runserver
```
